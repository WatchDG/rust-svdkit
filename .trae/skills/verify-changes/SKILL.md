# Проверка изменений

Навык проверяет изменения перегенерацией пакетов и запуском проверки примеров.

## Шаги проверки

1. **Перегенерация пакетов**: Для каждого пакета запустить:
   ```bash
   cargo run --bin svd2rs -- --svd_file tests/svds/<package>.svd --out_dir generated/<package>
   ```

   Список пакетов:
   - `nrf52840`

2. **Проверка примеров**:
   ```bash
   cargo check --examples
   ```

3. **Результат**: Убедиться, что нет ошибок в обоих шагах.

## Пример использования

```
Пользователь: проверь изменения
Агент: Вызывает навык verify-changes для перегенерации и валидации
```
